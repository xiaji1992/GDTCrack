# GDTCrack
腾讯广点通sdk(iOS)破解版（支持自定义包名）

主要功能：可以自定义包名，绕过errorCode=5006的错误（包名校验非法）

## 使用场景
用于多个iOS应用使用同一个广告ID

用过腾讯广点通的开发者都知道从2016年开始腾讯开始启动包名校验功能，这让很多之前几个app公用一个id的开发者很苦恼，于是就有了这个广告sdk破解版的开发，造福广大开发者（虽然来得有点迟）
